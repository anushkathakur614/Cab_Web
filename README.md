🚖 YourCab – Cab Booking Website

A modern and responsive Cab Booking Website developed to simplify online cab reservations for customers and help manage bookings efficiently. The project provides a smooth booking experience with real-time booking management, customer feedback, email notifications, and an admin dashboard.

---

✨ Features

👤 User Features

- Responsive and modern user interface
- Dark Mode support
- Cab Booking Form
- Automatic Booking ID Generation
- View My Bookings
- Update Booking Details
- Cancel Booking
- Customer Feedback System
- Email Confirmation after booking
- Email Notification after booking update and cancellation

🛠️ Admin Features

- Admin Dashboard
- Total Bookings
- Confirmed Bookings
- Pending Bookings
- Cancelled Bookings
- Recent Bookings Table
- Customer Feedback Display

🚀 How to Run the Project

1. Clone the repository.
2. Open the project in VS Code.
3. Install dependencies:

npm install

4. Start the backend:

npm start

5. Open the frontend using Live Server.

---

📧 Email Notifications

The system automatically sends email notifications for:

- New Booking
- Booking Update
- Booking Cancellation
- New Customer Feedback
